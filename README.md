# Advanced Table Component for Material UI
### This project is archived, no git history
This component has tons of data table features not found in base Material UI. Row sorting, reordering, filtering. Half a dozen input types with input validation. Select all. Fully customizable styles. And a lot more!

[You can play with a running example of this component here](https://danitheturtle.github.io/mui-advanced-table/examples/kitchen-sink/index.html)

Assumes that your theme.spacing function is either undefined or returns 'rem' units
